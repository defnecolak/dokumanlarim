# Security policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it responsibly.

- Contact: **ceylanatay@dokumanlarim.com**
- Also available at: `/.well-known/security.txt`

Please include:
- A clear description and impact
- Steps to reproduce (screenshots/requests if possible)
- Any proof-of-concept **without** accessing real user data

We will acknowledge receipt and work on a fix as quickly as possible.
